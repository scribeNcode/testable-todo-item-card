# testable-todo-item-card

HNG Internship Stage 0 Task

A  responsive to-do application designed with a focus on a seamless mobile experience. This project balances "moderate and classy" aesthetics with high-performance functionality.

---

## Features

- **Typography:** Uses the **Montserrat** Google Font for a professional, urban look.
- **Fully Accessible:** Programmatically linked labels, screen-reader friendly `datetime` attributes, and `aria-live` regions for status updates.
- **Keyboard Navigable:** checkboxes that maintain a clear focus-visible state
- **Mobile-First Design:** Features a full-width card layout.
- **Zero-Scroll Interface:** A locked viewport height ensures a "native app" feel on mobile devices by using `100dvh`.

---

##  Built With

- **HTML5:** Semantic markup including `<time>` and `<label>` for accessibility.
- **CSS3:** Custom Flexbox layouts, Media Queries, and a "visually hidden" utility class for screen readers.
- **Google Fonts:** Montserrat (Primary).
